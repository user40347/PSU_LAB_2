# Задание 1

## Задача 1

### Текст задачи

Получить список из первых цифр чисел, содержащихся в исходном списке

### Алгоритм решения

1) При помощи метода readPositiveInt запрашиваем ввод целого, положительного числа, получаем и проверяем, если введеная строка соответствует критериям, возвращаем в n, иначе повторяем запрос
2) С помощю метода readSelectedMethod запрашиваем выбор метода заполнения массива, проверяем, если это "0" или "1" - возвращаем значение в selectedMethod
3) Передаем в createList n и selected method, и исходя из выбранного кол-ва значений и выбранного метода заполнения инициализируем список
4) Выводим полученный список
5) При помощи List.map применяем функцию для получения первой цифры числа (takeFirst), результат записываем в resultList
6) Выводим новый, полученный список

## Задача 2

### Текст задачи

Сколько раз в списке встречается заданное число?

### Алгоритм решения

1) При помощи метода readPositiveInt запрашиваем ввод целого, положительного числа, получаем и проверяем, если введеная строка соответствует критериям, возвращаем в n, иначе повторяем запрос
2) С помощю метода readSelectedMethod запрашиваем выбор метода заполнения массива, проверяем, если это "0" или "1" - возвращаем значение в selectedMethod
3) Передаем в createList n и selected method, и исходя из выбранного кол-ва значений и выбранного метода заполнения инициализируем список
4) Выводим полученный список
5) При помощи метода readFloat запрашиваем ввод вещественного числа, получаем и проверяем, если введеная строка соответствует критериям, возвращаем в compnum, иначе повторяем запрос
6) Передаем в countOfMatches list и compnum для нахождения в list количества числел compnum. В countOfMatches используется List.fold для подсчета количества чисел соответствующих target (compnum). Полученный результат сохраняем в matches
7) Выводим количество чисел в списке, соответствующих заданному
